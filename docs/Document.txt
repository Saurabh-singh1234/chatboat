# LangChain + LangGraph + Hugging Face AI Agent

## Complete Beginner Project Documentation

## 1. Project Overview

In this project, you will build a **Professional AI Chat Agent** using:

* **LangChain** – Framework for building AI applications
* **LangGraph** – Controls the workflow of the AI agent
* **Hugging Face Transformers** – Provides the Large Language Model (LLM)
* **Streamlit** – Creates the web-based chat interface

This project does **not require an OpenAI API key**. The model runs using Hugging Face Transformers.

---

# 2. Project Goal

The chatbot should be able to:

* Answer general questions
* Explain programming concepts
* Answer AWS questions
* Help with writing
* Answer aptitude questions
* Explain AI and Machine Learning
* Hold professional conversations

# 7. model.py

```python
from transformers import pipeline

chatbot = pipeline(
    "text-generation",
    model="microsoft/DialoGPT-medium",
    max_new_tokens=150
)

def ask_llm(question):

    prompt = f"User: {question}\nAssistant:"

    result = chatbot(prompt)

    text = result[0]["generated_text"]

    answer = text.replace(prompt, "").strip()

    return answer
```

---

# 8. agent.py

```python
from typing import TypedDict
from langgraph.graph import StateGraph, END
from model import ask_llm

class AgentState(TypedDict):
    question: str
    answer: str

def chatbot_node(state: AgentState):

    answer = ask_llm(state["question"])

    return {
        "question": state["question"],
        "answer": answer
    }

graph = StateGraph(AgentState)

graph.add_node("chatbot", chatbot_node)

graph.set_entry_point("chatbot")

graph.add_edge("chatbot", END)

agent = graph.compile()
```

---

```

answer

# 9. app.py

```python
import streamlit as st
from agent import agent

st.set_page_config(
    page_title="Professional AI Agent",
    page_icon="🤖",
    layout="centered"
)

st.title("🤖 Professional AI Agent")

st.write("Ask me anything!")

if "messages" not in st.session_state:
    st.session_state.messages = []

for msg in st.session_state.messages:
    with st.chat_message(msg["role"]):
        st.markdown(msg["content"])

question = st.chat_input("Ask anything...")

if question:

    st.session_state.messages.append(
        {
            "role":"user",
            "content":question
        }
    )

    with st.chat_message("user"):
        st.markdown(question)

    result = agent.invoke(
        {
            "question":question,
            "answer":""
        }
    )

    answer = result["answer"]

    with st.chat_message("assistant"):
        st.markdown(answer)

    st.session_state.messages.append(
        {
            "role":"assistant",
            "content":answer
        }
    )
```

---

# 13. Current Limitation

The example uses **DialoGPT**, which is designed for conversational chat.

It is not the best choice for:

* Programming
* Mathematics
* Reasoning
* Technical explanations

---

# 14. Better Hugging Face Models

For higher-quality answers, replace DialoGPT with one of these instruction-tuned models:

* Qwen/Qwen2.5-1.5B-Instruct
* microsoft/Phi-3-mini-4k-instruct
* TinyLlama/TinyLlama-1.1B-Chat-v1.0

These models perform much better for coding, AWS, AI, mathematics, writing, and general question answering.
